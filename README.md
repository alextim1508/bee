# bee-rtc

Desktop приложение для управления детектором. Включает в себя функциональность для визуализации данных, 
снятия метрологических характеристик и конфигурации детектора.

## Стек технологий
- **Java 17**
- **JavaFX**
- **java.net. UDP/IP**
